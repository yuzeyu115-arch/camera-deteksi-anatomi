# camera-deteksi-anatomi
Aplikasi deteksi anatomi berbasis browser.

File utama: `camera_tracking_browser.html`

## Cara pakai
1. Upload semua file ke hosting statis atau domain dengan HTTPS.
2. Buka `camera_tracking_browser.html` di browser.
3. Klik tombol "Mulai Kamera" dan izinkan akses kamera.

## Catatan
- Proyek ini berjalan sepenuhnya dengan HTML/CSS/JavaScript.
- MediaPipe dimuat dari CDN, sehingga tidak perlu instalasi Python.
- `camera_tracking.js` berisi seluruh logika aplikasi.
- `run_server.ps1` hanya untuk pengujian lokal pada Windows.
